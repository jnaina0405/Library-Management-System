# Library Management System (C++)

## Overview
A menu-driven Library Management System developed in C++ that allows users to manage books, issue and return them, and store data persistently using file handling.

## Features
- Add new books
- View all books
- Issue and return books
- File-based data storage
- Object-oriented design

## Technologies Used
- C++
- OOP concepts
- File handling
- STL (vector, string)

## How to Run
1. Compile:
   g++ main.cpp -o library
2. Run:
   ./library
